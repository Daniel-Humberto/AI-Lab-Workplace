# AI-Lab-Workplace

Laboratorio basado en Docker, para automatizar y experimentar con LLMs locales usando Ollama y Open WebUI. Incluye orquestadores como N8N, LangChain, Apache AirFlow, infraestructura de monitoreo, bases de datos vectoriales y herramientas para fine-tuning.
